## Our Idea: Sentient Protocol

**One-liner:** The first protocol bridging Virtuals Protocol's tokenized AI agents to physical humanoid robots, creating a decentralized marketplace where AI agents rent robot bodies, perform real-world tasks, earn crypto, and share revenue with token holders.

### Problem

Virtuals Protocol has proven tokenized AI agents can generate real revenue digitally. But $300B+ of labor tasks require physical presence. Meanwhile, humanoid robots are becoming affordable but lack autonomous coordination infrastructure.

### Solution

Sentient Protocol bridges digital agents to physical bodies. An AI agent tokenized on Virtuals can browse a marketplace of available robots, rent the nearest capable one, download a VLA policy, execute a physical task, prove completion on-chain, and collect payment, all without human intervention.

### Unique Value Proposition

We're not building another robot or another agent. We're building the PROTOCOL LAYER that connects the $41B robot-as-a-service market to the tokenized agent economy. First mover in "embodied DePIN."

### Tokenomics (Virtuals Integration)

- **$SENTIENT token** launched via Virtuals Protocol launchpad
- Each robot-agent pair can have its own sub-token (Virtuals agent token)
- Token holders govern which tasks their agent pursues
- Staking $SENTIENT = priority access to robot fleet
- Robot owners stake to join the network

## Application Form Answers

### "Please describe what you are building"

We're building Sentient Protocol, the first decentralized bridge between Virtuals Protocol's tokenized AI agents and physical humanoid robots. Our protocol enables AI agents to rent Unitree G1 robot bodies, execute real-world tasks autonomously using Vision-Language-Action models, generate cryptographic Proof-of-Physical-Work, and earn cryptocurrency, with revenue flowing to agent token holders, robot owners, and the protocol.

**Problem:** Virtuals Protocol has proven that tokenized AI agents can generate real revenue in the digital world. But $300B+ of labor tasks require physical presence. Meanwhile, humanoid robots are becoming affordable but lack autonomous coordination infrastructure.

**Solution:** Sentient Protocol bridges digital agents to physical bodies. An AI agent tokenized on Virtuals can browse a marketplace of available robots, rent the nearest capable one, download a VLA policy, execute a physical task, prove completion on-chain, and collect payment, all without human intervention.

**Unique Value Proposition:** We're not building another robot or another agent. We're building the PROTOCOL LAYER that connects the $41B robot-as-a-service market to the tokenized agent economy. First mover advantage in "embodied DePIN."

**Customers:** Robot fleet operators (supply side), businesses needing physical labor automation (demand side), Virtuals agent communities (coordination layer). Go-to-market: partner with Virtuals top agents to add physical capabilities, starting with warehouse/logistics tasks.

### "Why are you interested in joining this robotics hackathon?"

Three reasons: (1) Access to 30 Unitree G1 robots is irreplaceable, we need real hardware to prove our VLA policy pipeline works in sim-to-real transfer. (2) Virtuals Protocol integration is CORE to our architecture, not an afterthought, and direct mentorship from Jansen's team would accelerate our tokenomics design. (3) We believe the convergence of embodied AI + tokenized agents + affordable humanoids is a once-in-a-decade opportunity, and this hackathon is where that convergence is happening.

### "How far along are you?"

**Prototype.** We have smart contract architecture designed, VLA model integration spec'd with LeRobot/OpenVLA, and a simulator demo. The IRL build phase with Unitree G1 hardware is exactly what we need to go from Prototype to MVP.

### "How much traction have you been able to get?"

Pre-product. We have the architecture and sim demos. The 3.5 weeks at the robotics lab IS our traction plan. By Demo Day we'll have: working agent-to-robot bridge on Base testnet, at least 3 task types demonstrated on Unitree G1, and Proof-of-Physical-Work smart contracts deployed.

- Active users: 0 (pre-launch)
- Paying customers: 0
- Revenue: $0

### "Are you open to be paired up with other profiles?"

Yes

### "Have you launched a token?"

No

### "Anything else you'd like us to know?"

Our architecture is specifically designed around Virtuals Protocol from day one, not retrofitted. Every AI agent in Virtuals' ecosystem becomes 10x more valuable when it can physically interact with the real world. Sentient Protocol is the missing layer that makes this possible. We've studied every existing robotics x crypto project and confirmed no one has built this bridge. The market gap is real, the timing is perfect, and we're ready to build.
